Key requirements
  • Minimum 16 characters — longer keys are significantly harder to brute-force
  • Any characters allowed — letters, digits, symbols, spaces (leading/trailing spaces are trimmed)
  • Verified via SHA-256 hash — print requests are authenticated by a one-way fingerprint; the plaintext key is never transmitted over any network
  • PIN-protected viewing — the key is stored encrypted with AES-256-GCM; reveal it using your account PIN; if you forget your PIN, clear and set a new key
